    Input Text              name    ${group_name}
    Input Text              description     This is test group
    Select Radio Button     permissions     1
    
    Click Button            Save

    Location Should Be      ${GROUPS URL}
    Page Should Contain     ${group_name}

    Click Button            Save
    Location Should Be      ${GROUPS URL}
    Page Should Contain     ${group_name}-Edited
