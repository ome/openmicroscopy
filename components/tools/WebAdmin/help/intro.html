1. Introduction

This document provides a brief guide to the Beta 2.2 version of the OMERO.webadmin software, released for general use in October 2007. The OMERO.webadmin tool provides a simple interface to administer an OMERO server.

The software has been developed as part of the Open Microscopy Environment (OME) consortium. For more information on OMERO, including information on bugs and updated versions, visit http://www.openmicroscopy.org

The rest of the guide is arranged as follows:

  • Section 2 - Glossary.
  • Section 3 - Configuration. Information about configuration.
  • Section 4 - Frequently Asked Question. How to use the tool.
We hope you enjoy using the software.
