4.17. How to edit an existing enumeration?

From the Enumerations page (see Figure 4.1-2), click on the Edit button in the Actions column.
Then modify the following required information:

  • Edit the enumeration's name: field Name .
Additional optional information can be updated:
  • Delete enumeration by the button .
