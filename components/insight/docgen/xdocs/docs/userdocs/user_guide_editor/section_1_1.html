Latest version of OMERO.editor is available for download from: http://trac.openmicroscopy.org.uk/ome/wiki/MilestoneDownloads

Once you have downloaded and unzipped the OMERO.editor package, double-click the application to launch.

OMERO.editor requires that you have Java 1.5 or higher installed. See http://www.java.com/en/ to download Java.
